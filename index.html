Language Tools

Some miscellaneous, somewhat throwaway language-related tools I've slapped together to help with some specific frustrations!

MSWord/Tab-Separated Table to HTML
May help with creating flash cards with tables (e.g. Conjugations/Declensions) in Mnemosyne, probably Anki, maybe others. Has some limits!
Word Language Tools Ribbon Tab VBA Add-in
An MS Word VBA Add-in that adds a new Ribbon Tab. Provides quick toggling on/off and viewing state of AutoCorrect, Proofing, Spelling, and Grammar settings which obscure diacritics and make improper corrections (especially correcting Ancient to Modern Greek). Also provides quick access to the table tools I use for glossing, and a few other things.
Word Macros (OLD, UNUSED)
(OLD, UNUSED) Word Macros to make things less painful. Currently includes a macro for toggling all auto-correction type features off/on, as well as grammar/spelling highlighting (which obscure diacritics)
